title: PPSSPP使用说明
---

{% note warn 注意 %}

1. 系统版本必须为Ubuntu 20.04
2. 只能运行在framebuffer下

{% endnote %}

## 升级系统

```sh
$ sudo apt update
$ sudo apt upgrade
$ sudo do-fenix-upgrade
```

## 安装

```sh
$ sudo apt update
$ sudo apt install ppsspp
```

## 运行

运行`launch_ppsspp.sh`就能运行起来模拟器

```sh
$ launch_ppsspp.sh
```
