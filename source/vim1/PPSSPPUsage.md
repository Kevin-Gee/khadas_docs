title: PPSSPP Instructions
---

{% note warn note %}

1. System version must be Ubuntu 20.04
2. Only run under framebuffer

{% endnote %}

## Upgrade System

```sh
$ sudo apt update
$ sudo apt upgrade
$ sudo do-fenix-upgrade
```

## Install

```sh
$ sudo apt update
$ sudo apt install ppsspp
```

## Run

Run `launch_ppsspp.sh` to run the PPSSPP

```sh
$ launch_ppsspp.sh
```
